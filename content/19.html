Exercise 19 (and Solution) Decode A Web Page Two

  Using the requests and BeautifulSoup Python libraries, print to the screen the full text of the article on this website: http://www.vanityfair.com/society/2014/06/monica-lewinsky-humiliation-culture.

  使用requests和BeautifulSoupPython庫，在此網站上將文章的全文打印到屏幕上：http : //www.vanityfair.com/society/2014/06/monica-lewinsky-humiliation-culture。

  The article is long, so it is split up between 4 pages. Your task is to print out the text to the screen so that you can read the full article without having to click any buttons.

  這篇文章很長，因此分為4頁。您的任務是將文本打印到屏幕上，以便無需單擊任何按鈕即可閱讀全文。

  (Hint: The post here describes in detail how to use the BeautifulSoup and requests libraries through the solution of the exercise posted here.)

  （提示：此處的帖子詳細介紹瞭如何通過此處發布的練習的解決方案使用BeautifulSoup和requests庫。）

  This will just print the full text of the article to the screen. It will not make it easy to read, so next exercise we will learn how to write this text to a .txt file.

  這只會將文章的全文打印到屏幕上。它不會使其易於閱讀，因此在下一個練習中，我們將學習如何將此文本寫入.txt文件。

Inspecting HTML on a web page  在網頁上檢查HTML

  If you have forgotten how to inspect the HTML of a web page, look at the detailed solution to the previous exercise for a very detailed explanation. In brief:

  如果您忘記瞭如何檢查網頁的HTML，請查看上一個練習的詳細解決方案，以獲取非常詳細的說明。簡單來說：

  1. Open the web page in Chrome.  在Chrome中打開網頁。
  2. Right-click the page and click “Inspect Element”   右鍵單擊頁面，然後單擊“檢查元素”
  3. Use the magnifying glass on the bottom-left of the page to click on elements of the page and look at their properties.   使用頁面左下方的放大鏡單擊頁面元素並查看其屬性。

Happy coding!   祝您編碼愉快！
